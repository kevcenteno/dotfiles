## Usage

### Mac

```
### Install homebrew and Git
ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go/install)"
brew doctor
brew install git
```

### Linux

```
### Install Git
sudo apt-get install git-core
```

### Common

```
git clone git@github.com:kevcenteno/dotfiles.git ~/dotfiles
cd ~/dotfiles
sh ./setup.sh
```
